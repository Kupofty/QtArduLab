## Project
QtArduLab is a lightweight Qt-based application designed for **sending, receiving, and monitoring serial data** when working with **Arduino** or any device using UART communication.  

Its main goal is to provide a clean and efficient GUI for debugging, testing, and developing embedded systems.


## Requirements
- Qt 6 (Widgets module)
- C++17 or higher
